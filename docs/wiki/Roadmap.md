# Roadmap

Open Scrapers Desk should keep evolving as both a PyQt desktop app and a Python-side integration layer.

## Next priorities

- add charts and dashboard cards for saved scraper outputs
- improve comparison views across multiple runs
- add scheduling, notification, and publishing helpers
- deepen the Discord bridge examples and channel-ready formatting recipes
- polish packaging, installer workflows, and release automation
- improve onboarding, troubleshooting, and accessibility
