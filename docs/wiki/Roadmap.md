# Roadmap

Open Scrapers Desk should keep evolving as both a PyQt desktop app and a Python-side integration layer.

## Completed in the latest expansion

- added dashboard cards and comparison views
- added alert webhook settings and command previews
- added preset-based Discord bridge examples
- added zip checksum packaging output
- improved onboarding and troubleshooting docs
- added prompt-driven Python bridge helpers

## Next priorities

- add recurring scheduled execution instead of previews alone
- deepen publish flows for notifications and webhooks
- deepen the Discord bridge examples, prompt-routing support, and channel-ready formatting recipes
- polish packaging, installer workflows, signing, and release automation
- improve onboarding, troubleshooting, and accessibility
