# Using The Desktop App

Overview tab:

- set toolkit path, node path, output directory, and default save format
- save named workspaces
- refresh backend status and source health

Run Scrapers tab:

- filter the catalog
- inspect scraper parameters
- queue selected runs, categories, or the full catalog
- review the pending job queue and live log

Results Library tab:

- scan saved outputs
- read summary views
- search records
- inspect record detail and metadata
