# Using The Desktop App

Overview tab:

- set toolkit path, node path, output directory, and default save format
- save named workspaces
- refresh backend status and source health
- configure alert webhooks and Discord formatting preset

Run Scrapers tab:

- filter the catalog
- inspect scraper parameters
- queue selected runs, categories, or the full catalog
- preview run and health commands for automation wiring
- review the pending job queue and live log

Results Library tab:

- scan saved outputs
- read dashboard-style summary views
- compare two selected result files
- search records
- inspect record detail and metadata

If you want automation outside the GUI, use the preview buttons in the Run tab to copy the exact toolkit commands that the desktop app would execute.
