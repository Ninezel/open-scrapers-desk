# Using the Desktop App

## Overview

Use this tab to configure paths and validate the backend.

## Run Scrapers

Use this tab to:

- select a scraper
- enter parameters
- choose a limit
- save output to JSON
- run single or multi-scraper jobs

## Results Library

Use this tab to:

- scan JSON outputs
- read normalized records
- filter records by keyword
- inspect metadata and links

## Logs & Help

Use this tab for quick instructions and troubleshooting notes.
