# Home

Welcome to the Open Scrapers Desk wiki.

Open Scrapers Desk is the desktop companion for the separate Open Scrapers Toolkit repository. It is designed for people who want to run scraper workflows and read collected data without having to stay in the terminal.

## Core idea

Two repos, two roles:

- `open-scrapers-toolkit`: reusable scraper engine and CLI
- `open-scrapers-desk`: desktop GUI and release packaging

That separation keeps both sides simpler and easier to maintain.
