# Publishing Checklist

Use this when turning the repo into a public GitHub project.

## Repository checklist

- README is up to date
- license is present
- contribution docs exist
- security policy exists
- issue templates exist
- workflow builds successfully

## Release checklist

- smoke test passes
- PyInstaller build succeeds
- toolkit connection tested against a real backend checkout
- screenshots or release notes prepared
- wiki pages published

## Suggested GitHub configuration

- public repository
- wiki enabled
- discussions optional
- releases enabled
- actions enabled

## Suggested first release assets

- zipped PyInstaller bundle
- installer if Inno Setup is used
- short quick-start PDF or link to wiki Home
