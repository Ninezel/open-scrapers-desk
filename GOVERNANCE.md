# Governance

Open Scrapers Desk is the desktop companion to the Open Scrapers Toolkit.

## Project goals

- make scraper data approachable for non-programmers
- keep the app easy to package and distribute
- avoid locking the desktop app to one fragile backend contract

## Decision rules

- UI changes should keep result reading simple and clear.
- Backend integration changes should remain transparent and documented.
- Release workflows should favor reproducibility over cleverness.
